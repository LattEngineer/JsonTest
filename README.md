# JsonTest
이것은 객체(Object) 직렬화 및 역직렬화에 대한 빠른 이해를 위해 만들어진 예시입니다.
해당 과정을 수행하려면 Gson 다음 코드를 넣어주셔야 합니다.

# pom.xml 수정
    <dependencies>
      <dependency>
        <groupId>com.google.code.gson</groupId>
        <artifactId>gson</artifactId>
        <version>2.7</version>
      </dependency>
    </dependencies>

# 문의 관련
해당 프로젝트는 예시이므로, 문의를 받지 않습니다.
